III глава - Front-end разработка

Адаптивная и кроссбраузерная вёрстка

Вёрстка - процесс написания web-сайта. Я выделил бы несколько простых программ для этого: Sublime Text, Atom, Visual Studio Code. Из всех я пользуюсь последним, но к выбору следует подходить сугубо по комфорту работы внутри какого-либо из них.

Адаптивная вёрстка - вёрстка, при которой web-сайт выглядит на стационарных и мобильных устройствах по-разному, но не меняя его контент.

Кроссбраузерная вёрстка - вёрстка, при которой web-сайт выглядит в разных браузерах одинаково.

Этап написания HyperText Markup Language (HTML)

Этот этап является фундаментальным при создании web-сайта. Его цель - наполнить страницы контентом.

Этап написания Cascading Style Sheets (CSS)

Не менее важный этап, чем HTML. Подключая CSS, Вы получите возможность стилизовать Ваш web-сайт. Например, создать на нём единую цветовую гамму или выбрать определённый шрифт.

Этап написания JavaScript (JS)

На данном этапе web-сайту задаётся интерактивность. Другими словами - определённые элементы будут иметь определённый сценарий действий. Например, отправка сообщения, вписанного в разделе обратной связи, на почту.

Этап написания React JS и React Native JS

React JavaScript и React Native JavaScript применяются для создания интерактивных пользовательских интерфейсов.
